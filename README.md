# Classificador de ncRNA (RNA não-codificante)

Projeto criado para o grupo PyLadies

## Descrição do Projeto 
Sistema em Python para classificação automática de RNAs não-codificantes (ncRNAs) baseado em características sequenciais como comprimento e conteúdo GC. Desenvolvido para avaliação formativa para aprendizado em bioinformática.

Objetivos
✅ Classificar ncRNAs por tamanho da sequência
✅ Calcular conteúdo GC e classificar por faixas
✅ Gerar relatórios da análise


## Como Usar
Pré-requisitos
Python 3.8 ou superior
Nenhuma dependência externa necessária
